## 4. 핵심 기능
- 아이템 매니저를 통해 획득 가능한 아이템 정보를 저장하고 관리합니다.
- JSON 기반의 세이브 시스템을 통해 플레이어 위치, 소지 아이템, 퍼즐 진행도, 기타 오브젝트 위치를 저장하고 불러올 수 있습니다.
- 각 퍼즐들은 설계된 구조에 맞게 작동합니다.
